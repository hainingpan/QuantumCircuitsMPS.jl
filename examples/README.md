# Examples

Four notebooks demonstrate the package's core use cases. Three are Julia
notebooks (run with the `Julia (Auto Threads) 1.12` kernel); one is a Python
notebook that post-processes data produced by one of the Julia notebooks.

| Notebook | Language | Purpose | Runtime hint |
|---|---|---|---|
| [`mipt_example.ipynb`](mipt_example.ipynb) | Julia | Measurement-Induced Phase Transition (MIPT): Haar-random gates vs. projective measurements, entanglement volume-law â†’ area-law crossover. | Minutes on a laptop (small `L`, few trajectories); phase-diagram sweeps scale up from there. |
| [`cipt_example.ipynb`](cipt_example.ipynb) | Julia | Control-Induced Phase Transition (CIPT): Reset/Haar staircase geometry, steady-state magnetization `Mz(p, L)`. Writes `cipt_Mz_data.csv` (ensemble-averaged `Mz_mean`/`Mz_sem` per `(p, L)`) for downstream finite-size-scaling analysis. | Several minutes to ~tens of minutes depending on ensemble size (`ensemble_size=500` in the shipped run) and `L` range. |
| [`cipt_fss.ipynb`](cipt_fss.ipynb) | **Python** | Finite-size scaling (FSS) data collapse of `cipt_Mz_data.csv` using the [FSS](https://github.com/hainingpan/FSS) package (`DataCollapse`), fitting `p_c`/`nu` and plotting raw + collapsed curves. | Seconds (fit is on a small ensemble-averaged CSV, no simulation). |
| [`AKLT_forcedmeas.ipynb`](AKLT_forcedmeas.ipynb) | Julia | Prepares the spin-1 AKLT state via repeated forced projection onto the `S=2` sector on pairs of sites; verifies string order / entanglement signatures. | Minutes on a laptop for moderate `L`. |

## Cross-language pipeline: `cipt_example.ipynb` â†’ `cipt_fss.ipynb`

`cipt_example.ipynb` and `cipt_fss.ipynb` form a two-stage, two-language
pipeline and must be run in this order:

1. **`cipt_example.ipynb`** (Julia, kernel `julia-_auto-threads_-1.12`) runs
   the CIPT simulation and writes `examples/cipt_Mz_data.csv` â€” a CSV with
   columns `p, L, Mz_mean, Mz_sem` (steady-state ensemble-averaged
   magnetization and its standard error, one row per `(p, L)` pair swept).
2. **`cipt_fss.ipynb`** (Python, kernel `python3`) reads
   `cipt_Mz_data.csv` and performs finite-size-scaling data collapse via the
   `fss` package's `DataCollapse` class, fitting the critical point `p_c` and
   correlation-length exponent `nu`.

`cipt_fss.ipynb` requires the `fss` Python package, which is **not on PyPI**
under that name â€” install it directly from GitHub:

```bash
pip install git+https://github.com/hainingpan/FSS.git
```

(`fss` also depends on `pandas`, `numpy`, `matplotlib`, `lmfit`, `joblib`,
`tqdm`, all pulled in automatically by the command above.)
